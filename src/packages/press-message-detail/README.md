  :msg-list="msgList"

## API

### Props


| 参数      | 说明           | 类型      | 默认值  |
| --------- | -------------- | --------- | ------- |
| list      | 消息列表       | _Array_   | `[]`    |
| loading   | 是否显示加载中 | _boolean_ | `false` |
| showInput | 是否显示输入框 | _boolean_ | `true`  |


### Event


| 事件名   | 说明     | 回调参数 |
| -------- | -------- | -------- |
| loadMore | 加载更多 | -        |
| send     | 发送消息 | 消息内容 |
